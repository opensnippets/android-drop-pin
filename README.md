# SimpleGps
## Overview
This android application demonstrates the use of GPS and Google Map

## Purpose
to provide a workable proof of concept and understand how to use GPS along with Google Map

## Tool
Kotlin/Android 4.1

## Test
Tested by Emulator and Nokia 5 (Android 9)
